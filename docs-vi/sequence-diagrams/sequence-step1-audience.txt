title: Bước 1 — Định nghĩa đối tượng mục tiêu

Web UI -> API Handler: Người dùng gửi prompt
API Handler -> Marketing Agent: Gọi qua AgentCore Runtime
Marketing Agent -> Databricks Agent: Ủy thác tác vụ đối tượng (A2A)
Databricks Agent -> MCP Gateway: Gọi các tool Databricks
MCP Gateway -> Databricks: Gọi API (SQL, khám phá catalog)
Databricks --> MCP Gateway: Kết quả
MCP Gateway --> Databricks Agent: Kết quả tool
Databricks Agent --> Marketing Agent: Dữ liệu đối tượng
Marketing Agent --> Web UI: Stream chi tiết đối tượng (SSE)

...: Người dùng rà soát và xác nhận đối tượng
